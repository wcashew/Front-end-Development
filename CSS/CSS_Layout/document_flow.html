inline元素特点：
1、不单独占一行
2、width、height不可以设置
3、width就是它包含的图片、文字的宽度
常见的元素：span、a
inline-block元素特点：
1、不单独占一行
2、元素的height、width、margin、padding都可以设置
常见的元素：img
